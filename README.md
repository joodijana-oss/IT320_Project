# Sanad

## Project Overview
Sanad is a web-based platform that helps patients find medication availability by connecting them with nearby pharmacies.

The system allows patients to submit medication requests and receive offers from pharmacies. Admins review requests before they become visible to pharmacies.

## Main Users
- Patient
- Pharmacy
- Admin

## Main Features

### Patient
- Register and log in
- Submit medication requests
- View submitted requests
- View request details
- View pharmacy offers

### Pharmacy
- View approved medication requests
- View request details
- Submit offers for available medications
- View submitted offers
- View reports

### Admin
- Review and approve/reject requests
- Manage users
- View system activity
